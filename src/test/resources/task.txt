Для сайта https://navisale.ru написать ui-автотест: добавление в корзину товаров с их проверками
 (добавить минимум 3 любых товара на выбор, каждый добавляется через переход по меню “Каталог”
 и открытие страницы с товаром, один из товаров добавляется в количестве > 1).
Для автотеста использовать maven/gradle, selenide, junit
(junit от 4 и выше, maven/gradle и selenide обязательно, для удобства написания автотестов дополнительно
можно использовать другие библиотеки). Результат прислать в виде ссылки на публичный репозиторий.
Проверки автотеста:
1) проверить что название и параметры товара, добавленные в корзину, соответствуют выбранному и добавленному ранее товару
(проверка названия, если есть возможность проверить цвет/размер - должна быть реализована проверка)
2) проверить общее количество выбранного товара в корзине с итоговым кол-вом в sidebar`е
(страница корзины, сравнить выбранный content в корзине с итоговым выводимым количеством товара в sidebar`е справа)
3) Для каждого товара сравнить ценник (ценник на странице товара с ценником в корзине)
4) Для страницы товара написать page object и выполнить проверки – сравнить название товара с его характеристиками
